Objetifos para se fazer na atividade

A atividade solicitada precisa seguir os seguintes passos: - Criar uma conta no site da github - fazer uma pasta e criar um arquivo html - fazer etições no arquivo - fazer um link com o arquivo no computador para a pasta do projeto no site do git hub.

Conhecimentos e vantagens adquiridas

Aprender a mexer no github, pois muitos projetos são utilizados por este meio, quando tem muitas pessoas trabalhando em um projeto é sempre importante manter todos atualizados e os arquivos salvos em um único lugar
